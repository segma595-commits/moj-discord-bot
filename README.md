# Moj Discord Bot

## Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/segma595-commits/moj-discord-bot.git
   ```
2. Navigate to the directory:
   ```bash
   cd moj-discord-bot
   ```
3. Install the required packages:
   ```bash
   npm install
   ```

## Usage
1. Create a `.env` file in the root directory and add your bot token:
   ```env
   DISCORD_TOKEN=your_bot_token_here
   ```
2. Start the bot:
   ```bash
   node index.js
   ```
3. Invite the bot to your server using the OAuth2 URL and enjoy!
